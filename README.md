# Open the Museum's Gates to Pirates — Paper Presentation

An HTML slide deck for the paper presentation of *Open the Museum's Gates to Pirates* (ZKM Center for Art and Media, iPRES 2018 — Heiss, Stricot & Vlaminck).

## Viewing

The presentation can be viewed [here](https://salmanassri.github.io/open_museum_gates_presentation.html).

Alternatively, open `open_museum_gates_presentation.html` in a web browser (double-click the file or use File → Open).

The deck needs a network connection for Google Fonts only.

## Navigation

- **Buttons:** PREV / NEXT at the bottom
- **Keyboard:** ← / ↑ for previous, → / ↓ / Space for next
- **Touch:** Swipe left (next) or right (previous)
